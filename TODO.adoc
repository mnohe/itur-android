== P1 — Blockers (must fix before the first commit)

=== Rotate exposed credentials

The `local.properties` file contains real, live secrets.
Even though it is listed in `.gitignore`, you must rotate them now because:

* They may already exist in IDE history, clipboard managers, or backup tools.
* Rotating them costs nothing; leaving them costs potential account compromise.

Actions:

* [ ] Revoke and regenerate the MapLibre/Mapbox token
  (`pk.eyJ1IjoibW5vaGUiLCJhIjoiY…`).
* [ ] Revoke and regenerate the Google OAuth Web Client ID
  (`913030919185-7nhme3d5liu37gpt2…`).
* [ ] Verify the Firebase API key in `app/google-services.json` is restricted
  to the correct Android package and SHA-1 fingerprint in the Firebase console.

== P3 — Medium priority (quality and completeness)
